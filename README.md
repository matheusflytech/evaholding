# Eva Holding — Site (Vite + React)

Site institucional da **Eva Holding** focado em automação e IA para empresas.

## Requisitos
- Node.js 18+ (recomendado)
- npm (ou outro gerenciador equivalente)

## Rodar localmente
```bash
npm install
npm run dev
```

## Build de produção
```bash
npm run build
npm run preview
```

## Estrutura
- `src/` — aplicação React
- `public/` — assets públicos (inclui `favicon.ico` e `og.png`)
